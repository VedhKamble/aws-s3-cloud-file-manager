# AWS S3 Cloud File Manager (Streamlit)

A cloud-based file management system built using Python, Streamlit, and AWS S3.
Users can upload, list, and download files from a private S3 bucket using a web interface.

## Features
- Upload files to AWS S3
- List stored files
- Download files securely
- Streamlit-based UI
- Private S3 bucket access

## Tech Stack
- Python
- Streamlit
- AWS S3
- boto3

## Cloud Concepts Learned
- IAM programmatic access
- AWS SDK integration
- Private cloud storage
- UI-to-cloud interaction
